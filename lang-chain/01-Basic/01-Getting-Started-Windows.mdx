## 安装git

[下载git](https://git-scm.com/download/win)

下载 64 位 Git 的 Windows 安装程序。

![git-download](./img/01-follow-the-installation-video-windows-01.png)

按照安装步骤确认并继续

![](./img/01-follow-the-installation-video-windows-02.png)

在后续所有步骤中均点击 Next 按钮以继续安装。

![](./img/01-follow-the-installation-video-windows-03.png)

按 Windows + R 键 - 必须以管理员身份运行 PowerShell

输入命令 "git" 并验证输出结果是否与下图所示一致

```powershell
git
```

![](./img/01-follow-the-installation-video-windows-04.png)

- 应用 PowerShell 策略

首先，以"管理员"身份运行 Windows PowerShell。

输入以下命令以应用该策略：

```powershell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser -Force
```

应用完成后，请关闭并重新启动 Windows PowerShell。在后续操作中，运行 Windows PowerShell 时均需选择"以管理员身份运行"。

## 安装pyenv

在安装 Python 前，请先安装 pyenv。pyenv 通过创建虚拟化环境来防止软件包之间发生冲突。

```bash
git clone https://github.com/pyenv-win/pyenv-win.git "$env:USERPROFILE\.pyenv"
```

- 添加环境变量

请复制并粘贴以下内容后执行。

```powershell
[System.Environment]::SetEnvironmentVariable('PYENV', $env:USERPROFILE + "\.pyenv\pyenv-win\", "User")
[System.Environment]::SetEnvironmentVariable('PYENV_ROOT', $env:USERPROFILE + "\.pyenv\pyenv-win\", "User")
[System.Environment]::SetEnvironmentVariable('PYENV_HOME', $env:USERPROFILE + "\.pyenv\pyenv-win\", "User")
```

请复制并粘贴以下内容后执行。

```powershell
[System.Environment]::SetEnvironmentVariable('PATH', $env:USERPROFILE + "\.pyenv\pyenv-win\bin;" + $env:USERPROFILE + "\.pyenv\pyenv-win\shims;" + [System.Environment]::GetEnvironmentVariable('PATH', "User"), "User")
```

请关闭当前 Windows PowerShell 并重新启动。

输入以下命令以验证其是否正常运行：

```powershell
pyenv
```

![](./img/01-follow-the-installation-video-windows-05.png)

- 安装 python

安装 Python 3.11

```powershell
pyenv install 3.11
```

将 Python 3.11 设置为默认运行时环境。

```powershell
pyenv global 3.11
```

检查 Python 版本

```powershell
python --version
```

请确保您已安装 3.11.9 版本（安装 3.11.11 版本亦可！）。

## 安装Poetry

请运行以下命令以安装 Poetry 包管理工具。

```bash
pip3 install poetry
```

- [Link to LangChain-OpenTutorial.git](https://github.com/LangChain-OpenTutorial/LangChain-OpenTutorial.git)

导航至 Documents 文件夹。

```powershell
cd ~/Documents
```

请执行以下命令以获取源代码。

```powershell
git clone https://github.com/LangChain-OpenTutorial/LangChain-OpenTutorial.git
```

请执行以下命令以切换到 LangChain-OpenTutorial 目录。

```powershell
cd LangChain-OpenTutorial
```

正在设置 Python 虚拟环境

```powershell
poetry shell
```

批量更新 Python 包

```powershell
poetry update
```

## 安装Jupyter扩展

搜索 "python" 并且安装

![](./img/01-follow-the-installation-video-windows-07.png)

搜索 "jupyter" 并完成安装
